# Discord Bot using Python
I made a discord server and a meme bot for it. It sends random meme from [this API](https://github.com/D3vd/Meme_Api) when the user sends '$meme' command, using [this link](https://discord.com/oauth2/authorize?client_id=1498917462136062003&permissions=2048&integration_type=0&scope=bot) you can invite the memebot to your server.

<img width="1839" height="861" alt="Screenshot (178)" src="https://github.com/user-attachments/assets/73b2e9ea-1d1d-4868-9064-f908dcbf2303" />

If you want to set up this bot then clone the repository and then change your current working directory to your repository.

```
git clone [https://github.com/Audi9602/Discord-Bot---Python.git](https://github.com/Audi9602/Discord-Bot---Python.git)
cd YOUR_GIT_REPO
```

Also, create a .env file to put your discord token and add it in .gitignore file then you can run the bot using bot.py file. (P.S. You need to have a Discord server with "Manager Server" permissions)
